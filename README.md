我当前要做 1 个 ai 项目示例：「实现 1 个交互窗口，接收用户的输入（表述或图片），调用模型能力反馈结果（ react 代码），通过调试模型的使用，尽量使结果更符合预期」。
技术选型使用：

- next.js
- Antd
- DeepSeek

1 个简单的封装了 openai 调用的项目，本质就是维护 1 个「对话历史」，拓展输入和输出的形式：

- 初始化时配置预设提示词
- 队列管理：添加、清理、维护上下文
- 格式转换：把各种输入（文字、图片、文件）转成 API 接受的格式
- 内容回显：同理
- 在这个基础上，再考虑如何精准表述：
  - 确保模型可以正确理解
  - 模型更少的消耗 token
